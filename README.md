# Vamsi Krapa — Portfolio

A modern, premium portfolio website built with pure HTML, CSS, and JavaScript. Designed for GitHub Pages deployment.

## Features
- Dark theme with gradient accents and glassmorphism
- Smooth scroll-reveal animations
- Parallax hero section with floating shapes
- Typing effect on hero motto
- Fully responsive (mobile-first)
- No dependencies — pure vanilla HTML/CSS/JS

## Sections
- Hero with CTA buttons
- About Me with highlights
- Professional Experience (timeline)
- Featured Projects
- Skills & Certifications
- Education & Research
- Contact & Social Links

## Deployment (GitHub Pages)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Select the branch (`main`) and root folder
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## Local Preview
Simply open `index.html` in your browser — no build step needed.
